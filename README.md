# node_julius
## Summary
* Node.js・Julius・OpenJTalkを使用した、喋ったことをオウム返ししてくれるアプリ

## Usage
* 使用する場合はJuliusをダウンロードし、moduleモードでlocalhost:8080を立てておいてください。
* 初回は```npm install```を行ってください。
* ```npm start```でアプリが起動します。
* マイクに向かって喋り、Juliusを終了させると、音声が帰ってきます(mei:normal)。
